## Installation

```bash
$ npm install --force
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
Este backend esta compilado con docker-compose, despues de hacer el git clone del repositorio debemos correr las migraciones con npm run migrations:run
